# MarketingOS Docs

Публичная база знаний приложения MarketingOS для маркетплейса Битрикс24.

## Назначение

Этот репозиторий хранит клиентскую документацию, которая будет опубликована на:

```text
https://app.rgdigital.ru/docs
```

Документация должна помогать пользователю:

- понять, что устанавливает приложение;
- быстро пройти первый запуск;
- разобраться в маркетинговом рабочем месте;
- начать работу с кампаниями, контентом и медиафайлами;
- найти ответы на частые вопросы;
- понять возможности первой версии приложения и направление развития системы.

## Принципы

- Пишем для пользователя, а не для разработчика.
- Не публикуем внутреннюю архитектуру, roadmap, agent workflow, runtime-ошибки и служебные данные.
- Не раскрываем данные donor-портала, токены, OAuth-настройки и внутренние детали разработки.
- Каждая статья должна отвечать на конкретный пользовательский вопрос.
- Сначала делаем полезную минимальную базу, потом расширяем.
