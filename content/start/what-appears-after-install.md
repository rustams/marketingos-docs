# Что появится в Bitrix24 после установки

После установки MarketingOS в Bitrix24 появится маркетинговое рабочее место с базовой структурой для управления процессом.

## Основные сущности

В первой версии используются три ключевых смарт-процесса:

1. Кампании
2. Контент
3. Медиафайлы

## Кампании

Кампания объединяет задачи, сроки, материалы, статусы и результаты маркетинговой активности.

## Контент

Карточка контента используется для подготовки отдельного материала: поста, статьи, видео, письма, вебинара или другого формата.

## Медиафайлы

Карточка медиафайла помогает учитывать изображения, видео, документы и другие материалы, которые используются в контенте.

## Базовая аналитика

В первой версии аналитика строится на данных, которые пользователь фиксирует вручную.

Это позволяет собрать базовую картину по выполнению и результатам кампании.
