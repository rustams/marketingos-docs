# Аналитика: обзор процесса

## Для чего эта статья

Статья объясняет, как в первой версии MarketingOS фиксировать базовые показатели и использовать их для оценки кампаний.

## Когда использовать

Откройте эту статью, если материал уже опубликован и нужно зафиксировать результат.

## Что входит в базовую аналитику

В первой версии можно учитывать:

- количество заявок;
- количество материалов в производстве;
- количество опубликованных материалов;
- выполнение сроков;
- просроченные задачи;
- базовые показатели по материалам;
- итоговые выводы по кампании.

## Как фиксируются данные

Показатели вносятся вручную в карточках материалов или кампаний.

Например:

- просмотры;
- переходы;
- заявки;
- реакции;
- другие показатели, которые использует ваша команда.

## Ограничения первой версии

В первой версии нет сквозной аналитики, аналитических панелей и автоматической интеграции с рекламными кабинетами.

## Результат

После фиксации показателей руководитель видит базовую картину по выполнению и результатам кампании.
