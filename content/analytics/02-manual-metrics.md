# Как фиксировать ручные показатели

## Для чего эта статья

Статья помогает внести базовые показатели по материалу или кампании.

## Когда использовать

Используйте её после публикации материала, когда появились первые данные.

## Какие показатели можно фиксировать

Например:

- просмотры;
- переходы;
- заявки;
- реакции;
- комментарии;
- другие показатели, которые использует ваша команда.

## Шаги

1. Откройте карточку материала или кампании.
2. Найдите блок с результатами или аналитикой.
3. Внесите доступные показатели.
4. Добавьте комментарий, если нужно пояснить результат.
5. Сохраните карточку.

## Результат

Показатели сохранены и доступны для оценки кампании.
