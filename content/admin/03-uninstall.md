# Удаление приложения

## Для чего эта статья

Статья помогает подготовиться к удалению MarketingOS с портала Bitrix24.

## Когда использовать

Используйте её перед удалением приложения из маркетплейса Bitrix24.

## Что проверить перед удалением

Проверьте:

- есть ли активные кампании;
- нужны ли сохранённые заявки на контент;
- используются ли материалы в производстве;
- нужны ли медиафайлы;
- нужно ли сохранить результаты кампаний.

## Что происходит при удалении

Поведение зависит от выбранного сценария удаления и версии приложения.

В общем случае приложение может быть удалено без автоматического удаления пользовательских данных.

## На что обратить внимание

Не удаляйте приложение во время активной работы команды с кампаниями и контентом без предварительного согласования внутри компании.

## Результат

Пользователь понимает, какие данные нужно проверить перед удалением приложения.
