---
title: "Как создать кампанию"
---

Создайте кампанию, когда нужно запланировать связанную маркетинговую активность: запуск продукта, рассылку, серию публикаций, мероприятие или рекламную активность.

Кампания станет верхним уровнем работы. К ней можно будет привязать заявки на контент, материалы в производстве, публикации и результаты.

## Перед началом

Подготовьте:

- цель кампании;
- период;
- ответственного;
- список основных материалов;
- результат, который нужно будет зафиксировать после публикации.

## Шаг 1. Откройте раздел «Кампании»

Перейдите в раздел «Кампании» в рабочем месте MarketingOS.

> Место для скриншота: список кампаний.

## Шаг 2. Создайте новую кампанию

Создайте новую карточку кампании.

> Место для скриншота: кнопка создания кампании.

## Шаг 3. Заполните основные данные

Укажите название кампании, цель, период, ответственного и краткое описание.

Название должно быть понятным для команды. Например: «Запуск весенней рассылки» или «Контент-план на июнь».

> Место для скриншота: карточка новой кампании с основными полями.

## Шаг 4. Проверьте описание

Описание должно помогать участникам понять, зачем проводится кампания и какие материалы нужно подготовить.

Не создавайте кампанию только как папку для материалов. Без цели, срока и ответственного её будет сложно контролировать и завершать.

## Шаг 5. Сохраните кампанию

После сохранения карточка кампании будет доступна для создания связанных заявок на контент.

## Что проверить

- Название понятно команде.
- Указаны цель и период.
- Назначен ответственный.
- Описание объясняет задачу кампании.
- Кампания готова для создания заявок на контент.
